# ВОР Генератор PRO — АйТима

## Переменные окружения на Railway

| Переменная | Значение |
|---|---|
| ANTHROPIC_API_KEY | sk-ant-... |
| YUKASSA_SHOP_ID | ваш shopId из ЮКассы |
| YUKASSA_SECRET_KEY | ваш secretKey из ЮКассы |
| BASE_URL | https://ваш-домен.railway.app |

## Вебхук ЮКасса

В личном кабинете ЮКассы → Настройки → HTTP-уведомления:
URL: https://ваш-домен.railway.app/webhook/yukassa
